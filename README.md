## bye
How to cleanly handle bye?
  1. Use `trap` in `BASH` for `SIGINT`, `SIGTERM`, `EXIT`
  2. Use `shutdown hook` in `Java`
  3. Use `atexit` in `Python`
  4. Use `at_exit` in `Ruby`
